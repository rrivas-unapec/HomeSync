# HomeSync

Sistema de gestión inmobiliaria para el alquiler y venta de propiedades.

## Estructura del proyecto

- `backend`: ASP.NET Core Web API.
- `frontend`: aplicación web pendiente de implementación.
- `docs`: documentación del proyecto.

## Tecnologías iniciales

- ASP.NET Core 8 Web API
- Swagger
- SQL Server y Entity Framework Core, pendientes de integración
- GitHub

## Ejecutar el backend localmente

Desde la carpeta raíz:

```bash
cd backend
dotnet restore HomeSync.sln
dotnet run --project HomeSync.Api
